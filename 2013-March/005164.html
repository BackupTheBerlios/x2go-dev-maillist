Severity: wishlist

When starting x2goclient with --broker-url, it isn't possible to preselect
a session using the --session=<session> argument. This would be a very nice
feature when using x2goclient on thin clients, as it seems unintuitive that
the user needs to choose a session when there is only one listed.
