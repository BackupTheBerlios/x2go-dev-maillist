If you are looking for a better performance for e.g. big flash movies,
my experience is that nothing helps, I've tried different videocards, a
faster connection, different compression types, etc.

I expect the bottleneck is a per-user resource limit in the NX
libraries. Maybe this limit is changeable, but I don't know how.

If a user has problems, in most cases another user on the same server
does not have problems.
