> And what is necessary for server part 

Only the package "x2goserver".

The packages x2goserver-home (for Ubuntu) and x2goserver-one (for
Debian) provide only an "easy setup script".

You can live without the administration packages and setup scripts.

> and which version you suppose to use for build like this (latest or older)?

I use "latest", and I don't see problems.

The future of the x2goclient-gtk client is unsure, it is likely that it
will be dropped. A new version of x2goclient (QT based) is developed at
the moment, with very interesting new futures like native session shadowing.
