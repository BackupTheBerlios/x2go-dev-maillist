http://code.x2go.org/gitweb?p=x2goserver.git;a=commitdiff;h=050bc008107fe2b4cb5a9f68f1bd14315306077f

The X2goDropbox provides an export spool mechanism (similar to the  
cups-x2go/x2goprint duo) for any MIME type you allow  
(client-side-wise) to be dropped into that X2goDropbox.

On the server-side you simply have to copy a file into $X2GO_DROPBOX  
which points to

   ~/.x2go/C-<session_id>/dropbox

The file will arrive on the client side (SSHFS) and the default  
application for this file will be opened (locally on the client).
