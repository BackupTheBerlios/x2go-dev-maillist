On 2011-04-27 09:55, Reinhard Tartler wrote:
>> I have added tags to X2go Git projects called ,,build-main''. Do you
>> > see any chance of building X2go packages on Launchpad that reflect the
>> > build-main tag state automatically?
> No.
>

This is not correct. Actually it is possible to make builds depend on a
tag, but there are probably lots of problems (tags not detected, only
master imported -> no tags on branches, etc). Therfore this seems to be
more complicated then setting up yet another git repo, and rebasing that
tag using a script and importing that repo into lp. Before doing so, we
should consider other solutions.
