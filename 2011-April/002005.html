Hi there

as there has been some talk about tmpfs on the list, I decided to start
the tmpfs branch on the official server. You can find it here:
http://code.x2go.org/gitweb?p=x2gothinclient.git;a=shortlog;h=refs/heads/tmpfs
The interesting folder is the x2gothinclientmanagement. There is a short
README explaining what the different scripts do.

Please remember, that this is work in progress and not very well tested.
None the less it should speedup your thin-client creation. Patches or
comments are very welcome.
