Hello Falk,

It is not save, but yes, it is possible.
You can specify pin codes for one or more gpg cards. Edit
~/.x2goclient/gnupg/pins:


#pincodes file
#Application ID                  PIN
D276000124010101000100000B630000 123456
D276000124010101000100000D9F0000 123456
