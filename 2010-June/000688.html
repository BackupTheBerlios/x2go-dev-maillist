Hello Gerry,

at the moment not all commandline options are available in the
properties file (*.x2go). We've already added the possibility of
autologin (using a key) and switches for the tool/status bar. We'll
include further options later.
The plugin demo site will contain different ideas how to use the plugin
on a website. No special javascript or css is needed for a basic site.

I've written about javascript because of the install mechanism. As
mozilla offers no plugin hosting at the moment, we'll need to host the
plugin on our server or mirrors. To guarantee that the plugin will be
installable the script will check different servers and should offer
some generic descriptions.

The properties file *.x2go can be placed somewhere in the net and it can
be generated by a cgi, servlet or php backend...

best regards,

Heinz
