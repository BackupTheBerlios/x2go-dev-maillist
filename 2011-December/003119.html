The problem seems to be that even though "x2goserver 
3.0.99.8-0~x2go-bpo1+lenny~main~388~build1 X2go server daemon scripts" 
is installed, "apt-get install x2goserver" claims it's already the 
newest version, when "x2goserver 
3.0.99.8-0~x2go1+squeeze~main~373~build1 X2go server daemon scripts" is 
available.

install x2goserver x2goserver-extensions" installs the newer version.
After that, I am able to install "apt-get install x2goserver-printing".

So while the problem regarding package installation seems to be solved 
(still not sure if printing works, though), I'm still curious as to what
