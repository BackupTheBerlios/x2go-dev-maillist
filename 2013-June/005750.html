in the meantime (between my initial mail in this thread and now) I  
have decided to create a release/4.0.1.x branch for X2Go Server in  
X2Go Git [1]. (Sorry!)

Please checkout the x2goserver code from this new branch and build it  
from source:

   git clone git://code.x2go.org/x2goserver.git
   cd x2goserver
   git fetch origin release/4.0.1.x:release/4.0.1.x
   git checkout release/4.0.1.x
   debuild -uc -us
   cd ..
   sudo dpkg -i x2goserver*.deb
