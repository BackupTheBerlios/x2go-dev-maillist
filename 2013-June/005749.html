> Hi to all who use X2Go Session Broker (x2gobroker.git),
>
> please test the latest builds (still building atm) of X2Go Session Broker
> and report back if you meet any sort of troubles. The session broker will
> require a latest version of X2Go Server (4.1.0.0-prerelease) or
> (4.0.0.3-prerelease).
>
> For testing:
>
>   git clone git://code.x2go.org/**x2gobroker.git<http://code.x2go.org/x2gobroker.git>
>   cd x2gobroker
>   debuild -uc -us
>   # install any build dependencies that are missing and re-run debuild -uc
> -us
>   cd ..
>
>   git clone git://code.x2go.org/**x2goserver.git<http://code.x2go.org/x2goserver.git>
>   cd x2goserver
>   git checkout release/4.0.0.x
>   debuild -uc -us
>   # install any build dependencies that are missing and re-run debuild -uc
> -us
>   cd ..
>
>   sudo dpkg -i *broker*.deb
>   sudo dpkg -i x2goserver*.deb
>
>   sudo apt-get install -f
>
> I am about to release x2gobroker 0.0.2.3 and do not want to cause any
> troubles in case some stuff may have become broken due to quite a code
> change I did to fix issue #152 [1] in X2Go BTS.
>
> Please test and report back. The release will ideally be on Friday if
> noone sends a veto.
>
> Thanks+Greets,
> Mike
