Package: x2gothinclient
Version: 1.0.1.8-0

A user "x2gothinclient" is established in the TCE- environment, but 
/usr/sbin/x2gothinclientd expects the user "x2goclient", with its home 
directory at /home/x2goclient. Therefore /usr/bin/x2goclient will not start, 
leaving the user in front of a black screen.

Intruducing the user 'x2goclient' solved the problem.
