Hi Alex,

for DirectRDP it might be really nice to make folder sharing  
configurable through the session profile manager tab ,,Folder Sharing''.

As RDP requires folder sharing to be set up with the session startup  
command whereas X2Go is able to load shared folders while the session  
has already been launched, my suggestion (draft) for local folder  
sharing with DirectRDP is this:

On session start:
   $ mkdir $HOME/.x2go/<session-id>/media

When providing a shared folder:
   $ ln -s $HOME/.x2go/<session-id>/media/_some_folder /<local>/<folder>/<path>

The rdesktop / xfreerdp calls are these:

   rdesktop -r disk:media:$HOME/.x2go/<session-id>/media <further-options>

   xfreerdp  --plugin rdpdr \
             --data disk:media:$HOME/.x2go/<session-id>/media -- \
             <further-options>
