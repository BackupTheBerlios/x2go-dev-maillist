Package: x2goclient
Version 3.99.3.0-prerelease
Severity: minor

Summary: x2goclient with DirectRDP on thin clients: regain control of  
the x2goclient window after end of DirectRDP session

Reproduce:

  o login to a desktop shell (e.g. KDE)
  o launch a terminal, inside the terminal type: x2goclient<ENTER>
  o arrange terminal and x2goclient window in a way so that both windows
    are visible
  o kill the window manager (e.g. killall kwin)
  o launch a DirectRDP session (no matter if with rdesktop or xfreerdp)
  o log out from the DirectRDP session
  o try to type somethin in the x2goclient window (e.g. try to re-login to the
    same session).
