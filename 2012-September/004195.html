Hi all,

next feature in X2Go server added: X2Go client-side shared folders now  
appear in RDP sessions if an X2Go proxied RDP session is used.

Windows 7 and 2008:
-------------------
In the folder navigation area of Windows Explorer search for

   media on <X2Go-Server-hostname>


Windows XP:
-----------
I have not tested this, but the documentation of rdesktop says: the  
share appears under

   \\tsclient\media

Happy testing (x2goserver from nightly builds is needed for testing)
