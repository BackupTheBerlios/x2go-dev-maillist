Hi all,

I played a little more with x2goclient, window focus, keyboard focus.

x2goclient is not able to gain keyboard focus in the main or in  
subwindows if no window manager is running.

Reproduce:

   o start a desktop shell (e.g. GNOME or KDE)
   o kill the window manager (killall metacity, killall compiz, killall kwin,
     etc.)
   o launch x2goclient via the application menu
   o in x2gclient try to add a new session profile
   o try to get focus for the session name textfield widget by clicking into it
     with the mouse
   o neither TAB nor Shift-TAB are working in the session profile  
manager window

Expected:

   o although that there is no window manager running, it should be  
able to gain
     keyboard control within the x2goclient GUI

Greets,
Mike
