Seconded.  I'm not sure if recent (10.6, 10.7) OS X versions are 
available for PPC, if they are, I guess we should support them, but as I 
already wrote in my poll, 10.5 is not receiving security updates any 
more, so the same goes for 10.4.  We should not waste time supporting 
seriously outdated versions.  Same goes for Windows: I haven't tried if 
the Windows X2Go Client runs on Windows 95, 98, Millenium, NT 
3.something, NT 4 or W2K.  Maybe it does, but I wouldn't rush to help 
anyone if it doesn't.

We should limit our support to operating systems still supported by 
their manufacturer/distributor.

Unless, of course, somebody proves me wrong by showing that a large part 
of our userbase is running on such manufacturer-unsupported operating 
systems.
