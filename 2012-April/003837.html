Hi all,

the first beta Release of the next PyHoca-GUI version has been  
uploaded to code.x2go.org:
http://code.x2go.org/releases/binary-win32/pyhoca-gui/pyhoca-gui_0.1.2.0-beta1_win32-setup.exe

If any of you on x2go-dev is interested in helping with the  
improvement of PyHoca-GUI, please feel free to grab the installer and  
give feedback on your testing.

The log file of PyHoca-GUI on Windows will be written to  
%APPDATA%\pyhoca-gui\.
There are many improvements and new features, only some may be named here:

   o published applications support
   o speed enhancement when clicking the GUI
   o runs and gets developed on Windows 7

Any kind of feedback is welcome!
