> 1) Published Applications - What is it exactly and how can I use it...

As the sponsor of this feature, I would describe it like this:
A feature similar to what Citrix offers as Published Applications, only 
as a Systray icon offering the application selection menu, and not as a 
part of Start/Programs or from within a web page (I believe that is/was 
called Citrix NFuse).

The advantage is that the configuration is server-sided, i.e. when you 
roll out the client configuration, you set it to "published 
applications", and when you wish to offer more applications, all you 
have to do is add their configuration file to the central directory on 
the server - you do not need to update the configuration on each client.
