I've attached a tarball with some files that are meant to be added to
the top level of the nx-libs redistributed source package.

They support the standard configure, make, make install commands for
installing everything that I thought might be useful. You can also use
autoconf to create a new configure script from configure.ac, if you
wish.

I'm not very experiences with autoconf, so my configuration is fairly
basic. It is probably missing a lot of checks that would be good, and
may have some included that it should not. But it works, and I think
it is better than nothing, which is what the package current has.

Feel free to make any changes you want before you add these files to
the package. And definitely at least have a look at them and give them
a try before adding them.
