What distro do you use? I am not an expert on x2goplugin and it  
actually lacks documentation, currently. About installation /  
compilation I can provide some knowledge:

On Debian/Ubuntu:

   1a. http://wiki.x2go.org/wiki:x2go-repository-debian
   1b. http://wiki.x2go.org/wiki:x2go-repository-ubuntu

Then

   $ apt-get update
   $ apt-get install x2goplugin


Other distros:

   git clone git://code.x2go.org/x2goclient.git
   git checkout 3.99.0.1
   make build_plugin && make install_plugin

On the first run, the build will probably fail because of missing  
dependencies. The build-dependencies on Debian are:

   debhelper (>= 7.0.50~),
   libqt4-dev,
   libldap2-dev,
   libssh-dev (>= 0.4.7),
   libcups2-dev,
   libx11-dev,
   doxygen,
   man2html-base | man2html
