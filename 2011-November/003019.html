On Debian or Ubuntu:

apt-get install devscripts

git clone git://code.x2go.org/x2goclient.git
cd x2goclient
debuild -uc -us

(you have to install the build-dependencies reported by dpkg-buildpackage).

On Other Linuxes:

git clone git://code.x2go.org/x2goclient.git
cd x2goclient
make && make install
