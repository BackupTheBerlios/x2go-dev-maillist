Hello,

hmm, I've had no problem installing x2go under 64-Bit 
Ubuntu/Lubuntu/Xubuntu/Ubuntu Studio from the nightly build ppa as 
described in the old wiki page.

1, add-apt-repository ppa:x2go/ppa
2. aptitude update
3. aptitude search x2go
4. Installing the necessary x2go components with aptitude install or 
apt-get install
