Hi all,

I have (over night) added MATE support to X2Go [1,2,3].

I esp. want to make the developers aware of a new project component:  
x2gomatebindings (a fork of the x2gognomebindings component) [4].

Due to lack of dependencies in Debian/Ubuntu, the x2gomatebindings  
package has to be built from source for now, packages are not in our  
.deb archives, yet.

The build dependencies for x2gomatebindings are atm fullfilled by  
packages from:

"""
$ cat /etc/apt/sources.list.d/mate.list
deb http://packages.mate-desktop.org/repo/debian wheezy main
deb-src http://packages.mate-desktop.org/repo/debian wheezy main
"""

Greets,
Mike
