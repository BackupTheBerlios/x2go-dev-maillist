You'll only need a "soundcard" on your client, but you'll need to make sure that every application using sound will be able to use your setup. If you choose pulseaudio for example, not every application uses the preferences of your desktop environment. Players like Amarok, Vlc or Xine do have their own audio backends with plugins, which also need to be configured to use pulseaudio.
