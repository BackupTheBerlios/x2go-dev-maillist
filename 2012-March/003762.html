> I would love to see mimebox included in x2goclient.  I know we had
> planned to write a "multimedia player" to automatically redirect to
> mimebox similar to the way printing works<snip>

If this ever happens, please make sure that it's a server-side feature 
that can be turned on or off e.g. by group membership or by some sort of 
config file option, preferably separate from printing.

There is a degree of security involved here - if it's too easy for a 
user to pass files between host and client, or if this even happens 
automatically behind the user's back, bad (TM) things can happen.

I prefer it the way that a user has to willingly and knowingly decide 
that a file is to be transferred between host and client.

While I do see that there are usage scenarios where security isn't an 
issue and such a mimebox/multimedia player thing might be neat, please 
leave the decision to the server administrator and not the user.

-Stefan
