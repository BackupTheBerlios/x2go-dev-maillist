Hi all,

for Debian squeeze (no Ubuntu yet, sorry) a new possible X2go Thin  
Client environment is ready for testing.

An installation howto is available in the X2go wiki:
http://wiki.x2go.org/installing_x2go_thin_client_environment_debian

I will be happy about feedback from those of you who try it out. At  
the beginning Morty and later on I have tried to simplify and automize  
many of the installation steps that formerly (Debian lenny) had to be  
done manually.

The code base of the current TCE can be found here:
http://code.x2go.org/gitweb?p=x2gothinclient.git;a=shortlog;h=refs/heads/tmpfs

Greets + Good luck,
Mike

the Git master branch of the x2gothinclient.git project.
