Hi all, hi eps. Heinz,

with some gCards you can observe Xorg/kbd freezes on X2go thin clients  
(I had a discussion with Heinz about that lately on the phone).

I have found a solution for this: use kernel boot option  
,,nomodeset''. The freeze is actually a graphical freeze and a result  
of interference between console fb and Xorg.

The latest X2go TCE packages have this boot option set by default.
