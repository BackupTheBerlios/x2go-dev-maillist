hi all,

today I have been working more on the x2goserver cleanup as described here:
http://wiki.x2go.org/roadmap?&#somewhen_work_in_progress

What I have done (in short) is:

   o x2goserver-compat package as a container to pick up all old x2goserver
     code (i.e. scripts that are not used by the recent clients anymore).
     x2goserver refers to this package with a Suggests: stanza.
   o scripts now get installed to

      $PREFIX/bin - scripts called from the client
      $PREFIX/sbin - scripts that are for the admin
      $PREFIX/lib/x2go - scripts called internally

   o moved man pages for scripts in $PREFIX/bin to man page section 1

   o to get rid of hard coded full paths I have added a script

      x2gobasepath

     that detects the installation PREFIX (/usr, /usr/local, etc.)

   o Tested the tarball installer...

      - make && make install -> works
      - make uninstall -> works first time
      - make uninstall -> ignores all errors if not installed
      - INSTALL howto updated

Greets,
