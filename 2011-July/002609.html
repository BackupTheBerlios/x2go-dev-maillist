Hi,

I'm not sure if this has been discussed previously, but PowerPC-support was removed in OS X Lion. Since x2go uses nxproxy, which is compiled on PPC, it fails to display the remote desktop (it connects and creates sessions just fine), yielding the following error in Finder:

"You can't open the application nxproxy because PowerPC applications are no longer supported."

A quick Google-search yields a FAQ at NoMachine[1] explaining the issue.

Trying to find a way to resolve this, I downloaded the NX 4 Preview for OS X, but the nxproxy isn't present in these packages.

Has anyone found a way to make x2go work in OS X Lion?
