At any rate, what needs to happen is that both OS's need to be able to
communicate with each other via ssh.
say for instance, your server is running on the host OS
(192.168.0.57). And your client is a guest OS in the same physical
machine but with bridged networking and get it's IP of 192.168.0.58.

In this setup, when connecting from the client app, you would connect
to 192.168.0.57, not localhost.
