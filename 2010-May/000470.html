The steps you've completed are required on the server side. If
everything worked fine, you should have a running and connectable
x2goserver on that machine.
You'll need further a client to connect to this server. This is normaly
another computer. On the client conputer you'll need to install
x2goclient. This is done with apt-get install x2goclient (you'll need to
setup the repository again too).
After the client installation you'll need to configure a session inside
x2goclient. For that you'll need the hostname or IP address of the
server, a username and its password (again on the server). This user
needs to be part of the group "x2gousers".
That's all you'll need to do to get x2go working.

I really hope this answered your question.
