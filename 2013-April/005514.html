On Mo 22 Apr 2013 17:12:42 CEST Mike Gabriel wrote:

> Hi All,
>
> I have added an automation mechanism for tagging bugs as pending. That is:
>
>   o fix a bug
>   o add your debian/changelog entry (as most of you know, the X2Go upstream
>     abuses debian/changelog as upstream changelog because of the  
> nice  changelog
>     tools in Debian)
>   o add (Fixes: #nnn) at the end of a changelog entry with #nnn as the bug
>     number you have fixed
>   o push your changes to git.x2go.org
>
> On the Git server there is a post-receive hook waiting for such   
> changelog entries and if one is spotted, it will generate a mail  
> likes  this [1].
>

> Mike
