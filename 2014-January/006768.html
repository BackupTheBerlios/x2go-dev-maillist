As Roderick Johnstone has noted in
https://bugzilla.redhat.com/show_bug.cgi?id=1033876, the NX code appears
to require a keymap.dir file to locate a keymap directory (see
NXVerifyXkbBaseDirectory in
nx-libs-3.5.0.21/nx-X11/programs/Xserver/xkb/ddxLoad.c).  This is no
longer required by even RHEL6 xorg code and so is not provided there or
in Fedora.

So it looks like I'm going to need to have the Fedora nxagent rpm
provide /usr/share/X11/xkb/keymap.dir.

Does this seem correct?
