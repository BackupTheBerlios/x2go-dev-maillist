Hi List & hi Bugwatchers,

Mike#1 and I discussed the following approach via phone today:

- x2goclient gets a new commandline option --keep-icon
- when this commandline option is set, the taskbar icon will not change 
upon successful connect (global setting)
- a new configfile option "keep-icon=true/false" will be added
- when this configfile option is set, the taskbar icon will not change 
upon successful connect (per-session setting)
- this change will be implemented for 4.0.1.3
- a config option/checkbox for it will be added with a later release.

- Stefan
