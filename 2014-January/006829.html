On 08/30/2013 02:06 PM, Mike Gabriel wrote:
> Hi all,
> 
> the libssh upstream developers released libssh 0.6.0rc1 some days ago
> [1].
> 
> Here is the list of new features that we hopefully find time and man
> power to cincorporate into X2Go Client:
> 
> Added new publicy key API. Added new userauth API. Added gssapi-mic
> userauth. ^^^^^^^^^^ -> Kerberos based logins
> 
> Added new callback based server API. Added Elliptic Curve DSA (ECDSA)
> support (with OpenSSL). Added Elliptic Curve Diffie Hellman (ECDH)
> support. ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^ -> finally ECDSA support
> 
> Added improved logging system. Added SSH-agent forwarding. 
> ^^^^^^^^^^^^^^^^^^^^ -> PyHoca-GUI already has this and it is cool!!! 
> Added key-reexchange. Improved documentation. Fixed timeout handling.
> 
> Greets, Mike
> 
> [1] http://www.libssh.org/2013/08/07/libssh-0-6-0rc1/
>
