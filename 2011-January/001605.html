Alex,
    For clients, there should be a default (Xming) that just installs if
the user does not do a "custom" install action. 

    If the user choses to do "custom" install then there should be a
choice to install whatever Xserver they like and pass a custom argument
string to it.

    For plugin, there should be choice of plugins that can be offered by
webserver.  One built w/Xming, the other w/VcXsrv.  That way either one,
or both could be offered to users.
