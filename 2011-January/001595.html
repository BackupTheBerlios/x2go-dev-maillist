Hello list,

I've applied the patch of Joachim Langenbach (thank you, Joachim) and
made some additional changes to x2goclient.

Now, you can configure x2goclient (Menu->options->settings) to:

- Display icon in system tray
- Hide to systray when minimized
- Hide to systray when closed
- Hide to systray after connection is established
- Restore from systemtray after session is disconnected

Using system tray icon you can:

- Iconify or restore client
- quit client
- suspend running session
- terminate running session
- export folder to running session

src,deb:
http://x2go.obviously-nice.de/deb/pool-heuler/x2goclient

apt:
deb http://x2go.obviously-nice.de/deb/ heuler main
