Good Afternoon,

I've created a small patch (Version 3.01-5, don't know which is current 
version) to integrate the first features of the system tray stuff. Currently 
this features of the list are supported:

 * Auto minimize to tray after established the connection successfully
 * Open the normal client

>  * Auto minimize to tray after established the connection successfully (and
> / or start already minimized?)
>  * Options right clicking on the tray icon:
>     * For each active connection:
>       * Close
>       * Pause
>       * Make directory available (In german version it is called "Ordner
> freigeben")
>     * Start a new connection
>  * Open the normal client

The patch includes some commented lines, they can be uncomment to display a 
standard message on minimize to the tray on first minimization. Therefore the 
system tray stuff can than be turned on by default (which is propably wanted by 
most users). Right now, system tray is deactivated and must be enabled at the 
ConfigDialog.

Regards,

Joachim Langenbach
