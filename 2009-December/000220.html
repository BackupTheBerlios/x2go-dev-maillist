I've got some remarks of a co-worker rearding the debian packages :

- sudo isn't a dependency of the x2goserver. So installing x2goserver
on a machine without sudo installed results in a non-workable sudoers
file
- the gpg-key of the repository is not on the website but the
repository itself is (what about placing the gpg key on the website ?)
- the entry being placed in /etc/apt/sources.list lacks the entry for
deb-src (debian style).
