Package: x2gothinclient
Version: 1.0.99.1
Severity: wishlist

Package creates /home/x2goclient and all the required subdirectories, if 
not present, but fails to create the user account and set its group 
membership.
This leads to an error about chown x2goclient:x2goclient 
(somedirectoryname) not being possible.

Current workaround: useradd x2goclient
(Note that this will create a locked account, as shown by the "!" in the 
password field in /etc/shadow)
