If you want to obtain the developers' keys and the X2go archive key
(Debian) then install

   apt-get install x2go-keyring

If you are using Ubuntu, use

   apt-add-repository ppa:x2go/stable

and--if you are behind a proxy--make sure that you have then env variables http_proxy _and_ https_proxy set to your webproxy server.
