Can you elaborate  a bit on what this does?

This sounds like it might fix the issue of x2go showing stock Xfce desktop
settings instead of Xubuntu-specific ones on Xubuntu.

Currently I'm working around this by having x2go call a custom script that
does "export XDG_CONFIG_

DIRS='/etc/xdg/xdg-xubuntu:/etc/xdg'" before the usual "exec /usr/share/

xubuntu/session.sh".
