Little/scarse howto:

   o install a chroot-server (next-server as in DHCP)
   o install x2gothinclientmanagement from packages.x2go.org/debian (heuler)
   o edit /etc/x2go/x2gothinclient_settings
   o run x2gothinclient_create
   o configure /var/lib/tftpboot properly (here you are on your own for now)
   o configure dhcpd
   o test-boot the client
