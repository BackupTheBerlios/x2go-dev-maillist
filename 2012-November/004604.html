There seem to be gtk related issues with GTK in Wheezy.
You'll get an error similar to

Gdk-WARNING **: The program 'gnome-control-center' received an X Window
System error.
This probably reflects a bug in the program.
The error was 'BadLength (poly request too large or internal Xlib length
erro'.
  (Details: serial 155 error_code 16 request_code 147 minor_code 1)

Currently known are Eclipse:
https://bugs.eclipse.org/bugs/show_bug.cgi?id=386955
https://wiki.archlinux.org/index.php/FreeNX#eclipse_crashes_when_editing_a_file


and the Gnome Control Center:
https://bugzilla.redhat.com/show_bug.cgi?id=827916
https://bugs.launchpad.net/ubuntu/+source/gnome-settings-daemon/+bug/867631
